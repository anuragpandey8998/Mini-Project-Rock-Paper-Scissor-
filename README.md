# 🎮 Rock Paper Scissors Game

A responsive and interactive **Rock Paper Scissors** web application developed using **HTML, CSS, and JavaScript**. This project demonstrates fundamental web development concepts including **DOM manipulation, event handling, game logic implementation, animations, and responsive design**.

The application allows users to play against the computer while tracking scores in real time through an engaging and user-friendly interface.

## 🚀 Features

* Interactive gameplay against the computer
* Real-time score tracking
* Randomized computer move generation
* Dynamic game result updates
* Smooth hover and click animations
* Modern and responsive user interface
* Floating visual effects for enhanced user experience
* Mobile-friendly design

## 🛠️ Tech Stack

* **HTML5** – Structure and semantic markup
* **CSS3** – Styling, animations, and responsiveness
* **JavaScript (ES6)** – Game logic and DOM manipulation

## ⚙️ How It Works

1. The player selects **Rock**, **Paper**, or **Scissors**.
2. The computer randomly generates its move.
3. The game compares both choices based on standard rules:

   * Rock defeats Scissors
   * Scissors defeats Paper
   * Paper defeats Rock
4. Scores are updated dynamically after every round.
5. The result is displayed instantly on the screen.
